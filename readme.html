AI Web Assistant - Kullanım Kılavuzu

1. Giriş

AI Web Assistant, kullanıcının girdiği görev (task) içeriğine göre web sayfalarında otomatik işlemler gerçekleştiren bir uygulamadır. Bu uygulama, OpenAI'nin GPT-4 modelini kullanarak web üzerinde belirlenen görevleri yerine getirir.

2. Gereksinimler

  • Python 3.x
  • OpenAI API Key
  • Aşağıdaki Python kütüphaneleri:
    • wxPython
    • asyncio
    • langchain_openai
    • dotenv
    • browser_use

3. Kurulum

3.1 OpenAI API Key Alma

OpenAI platformuna gidin: platform.openai.com
Giriş yapın ve API Key alın.
.env dosyasının içerisine aşağıdaki satırı ekleyin: 

OPENAI_API_KEY=your_api_key_here

4. Kurulum ve Çalıştırma

4.1 Paket Kurulumu

Gerekli paketleri kurmak için install.bat dosyasını çalıştırın:

install.bat

4.2 Uygulamayı Çalıştırma

Uygulamayı başlatmak için run.bat dosyasını çalıştırın:

run.bat

5. Arayüz ve Özellikler

5.1 Ana Menü ve Sekmeler

  • Agent: Görevleri yapılandırmak ve çalıştırmak için kullanılır.
  • Task History: Daha önce çalıştırılan görevlerin geçmişini gösterir.

5.2 Ana Araç Çubuğu

  • Run Task: Mevcut görevi çalıştırır.
  • Stop Task: Çalışan görevi durdurur.
  • Save: Görev yapılandırmasını kaydeder.
  • Load: Daha önce kaydedilmiş görev yapılandırmasını yükler.

6. Klavye Kısayolları

Kısayol İşlev
Ctrl+R Görevi çalıştırır
Ctrl+T Çalışan görevi durdurur
Ctrl+E Görev geçmişini dışa aktarır
Ctrl+S Görev yapılandırmasını kaydeder
Ctrl+O Görev yapılandırmasını yükler
Alt+1 Agent sekmesini görüntüler
Alt+2 History sekmesini görüntüler
Alt+F4 Uygulamadan çıkış yapar
